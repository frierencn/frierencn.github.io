1 搭建的基本工作

1.1 一些工具推荐

工欲善其事，必先利其器。

当你在搭建个人博客的过程中，有多处涉及到修改相关文件的内容，例如站点的配置文件 _config.yml ，里面的内容都是相关的代码，所有需要修改的这些文件你都可以通过Windows系统里自带的笔记本（Notepad）来打开，不过因为笔记本里显示的是纯文本，不同功能的代码也没有使用不同的颜色进行标记，所以阅读和查找起来比较麻烦。Notepad++是一款轻量级的文本编辑工具，可以看作是系统自带的笔记本的加强版，它支持语法高亮和语法折叠，很适合用来进行简单的代码编写和修改。

下载地址：Notepad++官网

此外，当你建成博客之后，你是在本地使用markdown格式来写文章的。Markdown是一种轻量级的标记语言，一大特点就是让你专心于你的写作而不是你文章的排版。只要掌握一些基本的常用语法，如各级标题的表示方法、加粗、斜体等，基本就可以进行文章写作了。下面这两篇文章可以帮助你了解什么是Markdown，以及如何使用Markdown。

Learning-Markdown (Markdown 入门参考)

Markdown 简明语法参考

你可以直接使用系统自带的笔记本或上面提到的Notepad++来进行写作，但专业的Markdown编辑器可能会给你更好的体验。我使用的是 Windows系统，Windows上比较少优秀的Markdown编辑器，我目前在用的、个人觉得非常不错的编辑器是 Typora。Typora 刚开始时仅有Mac端，不过现在也有Windows端的了。

下载地址：Typora — a minimal markdown editor, markdown reader.

2 进一步的完善工作：让你的博客更有个性

2.1 个性的主题

人靠衣装，佛靠金装。

进一步的完善工作主要集中于选择一个自己喜欢的主题，然后对该主题进行个性化的修改。主题就是你博客的外观。Hexo 的官网 为我们提供主题参考，在该网站中，如果你点击图片，你就会进入到使用了该主题的博客，它们是主题作者们的博客。如果你喜欢某一主题，你可以点击图片下面的主题名称，然后你会跳转到Github中主题作者存放该主题的目录下。前面的三篇文章都介绍了如何下载一个主题，我在这里遇到过一个问题：有的教程会在主题地址后面加上 .git ，有的教程后面没有加，我并不清楚这是为什么。

当你安装好自己喜欢的主题后，你可以使用前面提到的 hexo g 和 hexo s 来预览效果。接下来的很多工作就是对你所选择的主题进行修改，例如增加多一个菜单项，添加一个站内搜索功能等，此时你可以基于自己选择的主题到去搜索相关的教程。你可以在Github的该主题仓库里找到作者对该主题的介绍和使用，在 Issues 项下，你可以找到使用者们就某一问题或功能的讨论，那里是你寻找问题解决方法的好去处。

目前我的博客使用的主题是 NexT，其特点是简洁与大量留白，如果你也喜欢这一主题，那么下面这些网站或文章可能会帮助到你。【update-2017.10.15：目前我的博客已改用yilia主题~，其作者是Litten】【update-2019.03.19：目前博客主题已更换为Material Theme】【update-2020.04.23：目前博客主题已更换为Fluid】

2.2 自由的域名

我们建立的博客是托管在Github上的，所以我们博客的地址是Github的二级域名，例如，我的博客域名是 frierencn.github.io 。如果你希望自己的博客可以有一个自由个性的域名，你可以去域名提供商那里注册一个域名（当然是需要付费的）。域名提供商有不少，国内的有阿里云、腾讯云等，国外的有Godaddy（支持支付宝付款）等。至于是选择国内的域名提供商还是国外的，可以根据个人的需要来决定。以下两篇文章可以帮助你稍微了解一下国内和国外域名注册的区别：

月光博客：国内国外注册域名的对比

浅谈国内域名注册商与国外域名注册商的区别与优势

当你决定要注册一个域名的时候，你不仅要想域名的主体（如我的域名主体就是 frieren ），你还要选择一个域名后缀（如我的域名后缀就是 .me ，它其实是黑山共和国（Montenegro）的国家域名，就像 .cn 是中国的域名一样，但由于它和英语单词 me 一样，所以很多个人博客都拿它来当域名后缀了~)。常见的域名后缀有 .com , .net 等，国内的域名提供商会提供 .cn 后缀，但不建议使用~你可以去了解一下相关域名后缀的含义再作出决定也不迟。

当你注册好域名之后就需要进行DNS设置了。更多的介绍请阅读为部署在Github上的Hexo博客绑定个性域名【updated-2019.03.21: 该链接已失效】。当你为自己的博客绑定了一个自己的域名后，又有一些工作接踵而至了——你需要进行SEO优化，以便让搜索引擎更容易找到你的博客。关于这些内容以及其他更多个性化配置内容可以参考基于Hexo搭建个人博客——进阶篇(从入门到入土)。

4 后记

当你做完以上的工作后，你发现你的博客已经有模有样了。也许你并没有相关的基础，花了一天才把这些工作做好（就像我一样（。＾▽＾）），但是当你看到你通过“依葫芦画瓢”地敲了许多条命令、改了许多行代码、遇到许多问题然后又被你耐心地解决后建立起来的、每个图标和文字都饱含你的努力和折腾的个人博客时，你感受到的不是辛苦而是喜悦，甚至还会有一点自豪。你对待它就像你对待那些你细心栽培出来的花草一样，也许你觉得它对你而言是独一无二的，就像宇宙中的某朵玫瑰花对小王子而言是独一无二的一样。
